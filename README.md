# diabetes-risk-analysis
SQL and Tableau project analyzing diabetes risk factors across demographics, clinical metrics, and lifestyle factors.

# Diabetes Risk Analysis Dashboard

## Project Overview
- Which age group has the highest diabetes prevalence?
- How do BMI, HbA1c, and glucose levels between diabetic and non-diabetic patients?
- Are patients with hypertension or heart disease more likely to have diabetes?
- Does smoking history impact diabetes prevalence?
- How does diabetes risk vary by gender and age?

## Tools Used
- SQL (SQLite)
- Tableau Public
- Excel/CSV dataset

## Key Insights
- Diabetes prevalence increases with age, with patients age 60+ at highest risk
- Diabetic patients have higher BMI, HbA1c, and glucose levels
